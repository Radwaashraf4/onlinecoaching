# Online Coaching Website

A modern and responsive online coaching website built using **HTML**, **CSS**, and **Bootstrap**. The website is designed to provide an engaging platform for showcasing coaching services, courses, and contact information.

## ✨ Features

- Responsive design for all screen sizes
- Modern and clean user interface
- Home page with attractive hero section
- About section
- Services section
- Courses page
- Contact section
- Smooth navigation
- Bootstrap components
- Custom CSS styling
- Interactive JavaScript features
